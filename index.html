ID selectors #id_value { style properties } #identified <div id="identified"> Matches element based on value of element's id attribute, which must be unique in the document 1-0-0
Class selectors .class_name { style properties } .fancy <span class="fancy"> Matches elements based on contents of class attribute 0-1-0
Attribute selectors [attr=value] div[lang="en-us"] <div lang="en-us"> Matches elements based on presence or value of given attribute
Pseudo-classes selector:pseudo-class { property: value; } button:hover N/A Keyword added to selector (referred to as anchor element) that specifies special state of selected element(s)
Type selectors element { style properties } h1 { } <h1> Matches elements by node name, meaning selects all elements of given type within document 0-0-1
Pseudo-elements selector::pseudo-element { property: value; } a::before N/A Keyword added to selector that allows styling specific part of selected element
